This guide will walk you through creating one of the most valuable workflows: an automatic WhatsApp order confirmation for your Lightfunnels store.

## Prerequisites

Before you begin, make sure you have completed the following steps:

<CardGrid>
  <Card>
    #### 1. Connect a WhatsApp Device
    You need an active device to send messages from.
    [See Guide](./1.connecting-your-device)
  </Card>
  <Card>
    #### 2. Create a Message Template
    You need a template for your confirmation message.
    [See Guide](./2.creating-message-templates)
  </Card>
  <Card>
    #### 3. Connect Lightfunnels
    Link your Lightfunnels store from the **Connections** page.
  </Card>
</CardGrid>

<div class="steps">
### Step 1: Select the Automation Template
Navigate to **"Automations"** > **"New Automation"**. From the list, select **"Send WhatsApp on Lightfunnels Order"**.

### Step 2: Configure the Trigger
This section tells the automation **when** to run.
- **Lightfunnels Connection:** Select the store you connected earlier.
- **Funnel:** Choose the specific funnel to monitor for new orders.

### Step 3: Configure the Action
This section tells the automation **what** to do.
- **WhatsApp Device:** Select the device to send the message from.
- **Message Template:** Select your order confirmation template.

### Step 4: Create the Automation
Review your settings. If everything looks correct, click the **"Create Automation"** button.
</div>

That's it! Your automation is now active. The next time a customer places an order, they will instantly receive your personalized WhatsApp confirmation.
